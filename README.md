# Lucky Box: Japanese Raffle Simulator 📦🎫

An authentic digital simulation of the classic **Japanese triangular paper lot raffle**, built with **Flutter**. This app recreates the suspense of a physical raffle box, where each drawn lot is removed from the box, increasing the excitement as the remaining prizes decrease.

## 🌟 Key Features
- **Authentic Depletion Logic:** Just like a real raffle, drawn lots are removed from the box, changing the probability for each subsequent draw.
- **Simple Prize Registration:** Easily set up prizes using a `Quantity:Name` format (e.g., `1:Car`, `3:Bicycle`).
- **Visual Suspense:** Experience the thrill of "Obi-iwai" and traditional Japanese festival prize booths.
- **Global Support:** Fully localized in **50+ languages**, bringing a unique Japanese tradition to the world.

## 💡 Best For...
- **Parties & Events:** A portable and fair way to handle lucky draws and giveaways.
- **Classrooms:** Gamifying student selection or reward distribution.
- **Live Streaming:** Engaging viewers with a transparent and visual lottery system.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Real-time array manipulation and state management to handle the dynamic reduction of lots in the raffle pool.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
